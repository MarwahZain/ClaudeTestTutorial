# Customer Login

As a registered ParaBank customer, I want to log in with my username and password,
so that I can access my account services.

## Acceptance Criteria
- User can enter Username and Password on the homepage login form
- On valid credentials, user is redirected to the Accounts Overview page
- On invalid username or password, user sees an error message and remains
  on the login page
- Empty username or password fields are rejected with a validation message
